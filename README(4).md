# Simple Interest Calculator

This project contains a Bash script that calculates simple interest based on values entered by the user.

The calculator uses the following formula:

Simple Interest = (Principal × Rate × Time) / 100

## Input

The script asks the user to enter:

- Principal amount
- Annual rate of interest
- Time period in years

## Output

The program calculates and displays the simple interest.

## Project Files

- `simple-interest.sh` — Bash script for calculating simple interest

## Usage

Run the script from a Bash terminal:

```bash
bash simple-interest.sh
```

Follow the prompts and enter the principal amount, interest rate, and time period.
